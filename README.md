# M Shop - Zest Edition 🍋

A modern, high-end e-commerce concept built with **React**, **Redux Toolkit**, and **CSS Modules**. 

## ✨ Features
- **Luxury UI/UX:** Inspired by the "Zest" design system with rounded capsule elements.
- **State Management:** Fully handled by Redux Toolkit (Cart, Products).
- **Responsive Design:** Optimized for Mobile, Tablet, and Desktop.
- **Dynamic Search & Filter:** Advanced product filtering.

## 🛠️ Technologies
- React.js
- Redux Toolkit
- React Router DOM
- CSS Modules
- React Icons (Pi, Md, Hi)

## 🚀 How to Run
1. `npm install`
2. `npm run dev`
